StrykeX Dashboard
A modern, responsive financial and trading dashboard built with React and Vite. This project features a modular UI, sidebar navigation with dynamic SVG icons, and is optimized for production deployment on Netlify.

Live Demo
https://skrykex.netlify.app/

Tech Stack
Framework: React

Build Tool: Vite

Deployment: Netlify

Icons: SVG

Features
Fast Build: Optimized using Vite for instant hot module replacement.

Static Assets: Reliable icon rendering via the public directory for production stability.

CI/CD: Automatic deployments via GitHub and Netlify integration.

Installation
Clone the repo: git clone https://github.com/your-username/your-repo-name.git

Install dependencies: npm install

Start Development: npm run dev

Production Build: npm run build

Structure
/public/icons: Static SVG assets.

/src/components: React UI components.

/src/assets: Styles and global assets.
